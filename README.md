# 🚀 PiasX Hub

A modern personal dashboard website for managing APK files, tools, and quick-access data.

---

## ✨ Features

* 📁 Folder-based system (WhatsApp Tools, Copy Tools, Contact, etc.)
* 📥 1-click APK download
* 📋 One-click copy system (bKash, Nagad, Crypto)
* 🔗 Social media quick access (open directly)
* 🌑 Clean dark UI dashboard
* ⚡ Fast and lightweight (no backend required)

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Dark UI)
* JavaScript (Dynamic loading)
* JSON (Data management)

---

## 📂 Project Structure

```
/piasx-site
 ├── index.html
 ├── style.css
 ├── app.js
 ├── /data
 │     ├── whatsapp.json
 │     ├── copy.json
 │     ├── contact.json
 └── netlify.toml
```

---

## 🚀 Deployment

This project is deployed using:

* Netlify (hosting)
* Cloudflare (domain & DNS)
* GitHub (code management)

## ⚙️ How to Use

1. Clone this repo
2. Edit `/data` JSON files
3. Add your APK links / data
4. Deploy via Netlify

## 🔒 Notes

* APK files are hosted externally (Google Drive / Cloud)
